# UI/UX improvement plan

作成日: 2026-05-20

対象ブランチ: `UI-UX改善`

参考: [Zenn](https://zenn.dev/)

## 目的

学校の目安箱アプリを、投稿しやすく、意見を探しやすく、管理しやすい画面に改善する。
見た目だけを整えるのではなく、ログイン、投稿、閲覧、いいね、管理の流れが迷わず使える状態を目指す。

## Zenn から参考にする点

Zenn のトップページは、最初に `Trending` と `Explore` のような主要導線を出し、その下で `Tech`、`Ideas`、`Books`、`Featured` のようにコンテンツを分類している。
この構造は、利用者が「今見るべきもの」と「探したいもの」をすぐ切り替えられる点が参考になる。

このアプリでは、Zenn のデザインをそのままコピーせず、以下の考え方だけ取り入れる。

- 主要操作を画面上部に固定する: 投稿、閲覧、ログイン状態がすぐ分かるようにする。
- 一覧を分類する: 人気順、新着順、話題順を見出しとカードで整理する。
- カードの情報量を揃える: 投稿本文、出どころ、いいね数、操作を同じ位置に置く。
- 余白と線を控えめに使う: 学校向けなので、派手な装飾より読みやすさを優先する。
- 「もっと見る」の導線を明確にする: 一覧の続きがあることをボタンとして分かりやすくする。

## 現状の課題

- `index.html` にインライン CSS が多く、`style.css` と見た目の責任範囲が分かれていない。
- 投稿にはログインが必要だが、利用者画面にログインボタンや状態表示がない。
- タブ、投稿フォーム、一覧カード、戻るボタンの見た目が統一されていない。
- カイル表示が右上固定で、画面サイズによって本文や操作に重なる可能性がある。
- 投稿一覧の読み込み中、空状態、取得失敗時の表示が弱い。
- 管理画面は操作できるが、一覧、選択、削除の危険度が視覚的に分かりにくい。

## 改善方針

### 1. 情報設計

- 画面上部にアプリ名、短い説明、ログイン状態、主要タブをまとめる。
- 投稿フォームと閲覧一覧を同じ画面に置き、投稿後すぐ下の一覧へ移動できるようにする。
- 閲覧画面は Zenn のカテゴリ表示を参考に、人気、新着、話題を見出し付きセクションにする。

### 2. 投稿体験

- 投稿欄の説明文を短くし、何を書けばよいか分かるプレースホルダーにする。
- 未ログイン時は投稿ボタンの近くにログインが必要なことを表示する。
- 送信中はボタンを無効化し、二重送信を防ぐ。
- 投稿成功後は閲覧画面へ移動し、投稿されたことが分かる状態にする。

### 3. 閲覧体験

- 投稿カードは本文、種別、いいね数、操作ボタンを一貫した配置にする。
- トップ3表示は横スクロールに頼りすぎず、スマホでも読めるグリッドにする。
- 詳細一覧では戻る導線を分かりやすくし、現在見ているカテゴリ名を明確にする。
- 空状態、読み込み中、エラー状態を用意する。

### 4. 見た目

- ベースは白と薄いグレーで読みやすくする。
- アクセントカラーは青緑系を中心にし、学校向けの落ち着いた印象にする。
- ボタン、カード、入力欄の角丸、余白、影を統一する。
- 絵文字だけに頼らず、テキストでも意味が分かるボタンにする。

### 5. カイル表示

- 画面を邪魔しない位置とサイズに調整する。
- スマホでは固定表示を弱めるか、投稿フォームの近くに収める。
- 吹き出しは本文と重ならないようにする。

## 実装順

1. `index.html` の構造整理
   - インライン CSS を減らす。
   - ヘッダー、タブ、投稿フォーム、一覧、詳細を意味のあるブロックに分ける。

2. `style.css` へのスタイル集約
   - カラー、余白、カード、ボタン、フォーム、レスポンシブを定義する。
   - 現在表示されていない古いスタイルを整理する。

3. `app.js` の表示状態改善
   - ログイン状態、読み込み中、空状態、エラー状態の表示を追加する。
   - 投稿カード生成時のインライン style をやめる。

4. カイル UI の調整
   - PC とスマホで重なりにくい配置にする。
   - 投稿体験を邪魔しないサイズにする。

5. 動作確認
   - ローカル表示確認。
   - 投稿、閲覧、詳細表示、いいねの主要導線を確認する。
   - スマホ幅で重なりや文字切れがないか確認する。

## 完了条件

- 投稿画面で、未ログイン時とログイン時の状態が分かる。
- 閲覧画面で、人気順、新着順、話題順のカードが読みやすい。
- 詳細一覧へ移動して戻れる。
- いいねボタンの意味と数が分かる。
- カイル表示が主要操作に重ならない。
- PC 幅とスマホ幅の両方で文字やボタンがはみ出さない。

## 今回はまだやらないこと

- API の仕様変更。
- Firebase の設定変更。
- 投稿データ構造の変更。
- AI フィルターの再有効化。
- Zenn の見た目の完全再現。
- `admin.html` の変更。
